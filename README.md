# 🏥 Clinic-Platform | المنصة الطبية المتكاملة

> **حل رقمي ذكي لإدارة العيادات والمراكز الطبية، يهدف لتنظيم المواعيد وسجلات المرضى بأسلوب عصري وسريع.**

---

## ✨ المميزات الرئيسية

- **📅 نظام حجز ذكي**: واجهة سهلة للطلاب والمرضى لاختيار المواعيد المتاحة.
- **📂 السجلات الطبية**: إدارة ملفات المرضى والتاريخ المرضي بشكل منظم.
- **⚡ أداء سريع**: مبني باستخدام **Vite** و **React** لضمان أفضل تجربة مستخدم.
- **🎨 واجهة احترافية**: تصميم متوافق مع كافة الشاشات باستخدام **Tailwind CSS**.

---

## 🛠 التقنيات (Tech Stack)

- **Frontend**: React.js & Vite.
- **Styling**: Tailwind CSS & Framer Motion.
- **Components**: Radix UI & Lucide Icons.
- **Icons**: Lucide React.

---
