
# Waslet Khair - وصلة خير

منصة تبرعات خيرية تهدف إلى تسهيل عملية التبرع وربط المتبرعين بالجمعيات والحالات المحتاجة بطريقة أكثر شفافية وثقة.

---

## فكرة المشروع

وصلة خير هي منصة تجمع الجمعيات الخيرية في مكان واحد، وتتيح لكل جمعية عرض الحالات التي تحتاج إلى تبرعات، بحيث يستطيع المتبرع تصفح الحالات، معرفة تفاصيل كل حالة، اختيار الحالة التي يريد دعمها، والتبرع لها بشكل مباشر.

الفكرة الأساسية من المشروع هي حل مشاكل شائعة في التبرعات مثل:

- عدم معرفة المتبرع أين تذهب أمواله.
- ضعف الثقة في الحالات المنتشرة على السوشيال ميديا.
- صعوبة الوصول إلى جمعيات موثوقة في مكان واحد.
- عدم وجود متابعة واضحة لحالة التبرع.
- تشتت الحالات بين صفحات ومصادر كثيرة.

---

## الهدف من المشروع

يهدف المشروع إلى بناء منصة خيرية منظمة وموثوقة تساعد على:

- عرض الحالات الخيرية بشكل واضح ومنظم.
- تمكين المتبرع من اختيار الحالة التي يريد دعمها.
- توفير شفافية أكبر في عملية التبرع.
- مساعدة الجمعيات على نشر الحالات وإدارتها.
- إرسال إشعارات للمتبرعين عند حدوث تحديثات مهمة.
- تقديم تجربة استخدام سهلة وسريعة ومتجاوبة مع جميع الأجهزة.

---

## المستخدمون داخل النظام

المشروع يحتوي على أكثر من نوع مستخدم، ولكل نوع صلاحيات وتجربة مختلفة.

### 1. المتبرع - Donor

المتبرع يستطيع:

- إنشاء حساب جديد.
- تسجيل الدخول.
- تصفح الحالات.
- تصفح الجمعيات.
- مشاهدة تفاصيل كل حالة.
- التبرع لحالة محددة.
- إضافة حالات إلى المفضلة.
- متابعة الحالات المفضلة.
- استقبال إشعارات عند تحديث الحالة.
- مشاهدة سجل التبرعات الخاصة به.
- تعديل بيانات الحساب الشخصي.

---

### 2. الجمعية - Charity

الجمعية تستطيع:

- تسجيل الدخول إلى لوحة التحكم.
- إضافة حالات جديدة.
- تعديل بيانات الحالات.
- متابعة التبرعات الخاصة بالحالات.
- معرفة نسبة اكتمال كل حالة.
- إدارة بيانات الجمعية.
- مشاهدة الحالات التابعة لها.
- متابعة الإشعارات والتنبيهات.

---

### 3. الأدمن - Admin

الأدمن مسؤول عن إدارة المنصة بالكامل، ويستطيع:

- إدارة المستخدمين.
- إدارة الجمعيات.
- مراجعة الحالات.
- إدارة التبرعات.
- متابعة الإحصائيات العامة.
- إدارة التصنيفات.
- متابعة البلاغات أو المشاكل.
- التحكم في محتوى المنصة.
- متابعة النشاط العام داخل النظام.

---

## أهم مميزات المشروع

- واجهة عربية كاملة RTL.
- تصميم Responsive يعمل على الموبايل والتابلت والديسكتوب.
- دعم الوضع الليلي Dark Mode.
- نظام تسجيل دخول وإنشاء حساب.
- عرض الحالات الخيرية بطريقة منظمة.
- صفحة تفاصيل كاملة لكل حالة.
- نظام تبرع لحالة محددة.
- نظام مفضلة للحالات.
- إشعارات للمتبرعين.
- لوحة تحكم للإدارة.
- لوحة تحكم للجمعيات.
- سجل تبرعات للمتبرع.
- تصنيفات للحالات.
- فلترة وبحث داخل الحالات.
- واجهة حديثة باستخدام React.
- إدارة بيانات باستخدام React Query.
- تصميم UI احترافي باستخدام Tailwind CSS.
- أيقونات من Lucide React.
- قابلية التوسع مستقبلًا.

### Frontend

- React.js
- React Router DOM
- React Query / TanStack Query
- Axios
- Tailwind CSS
- Lucide React
- Framer Motion
- SweetAlert / Custom Alerts

---

### Backend

حسب نسخة المشروع، يمكن أن يكون الـ Backend مبنيًا باستخدام:

- ASP.NET Core Web API  
أو
- Node.js / Express.js

ويكون مسؤولًا عن:

- إدارة المستخدمين.
- تسجيل الدخول والتسجيل.
- إدارة الحالات.
- إدارة الجمعيات.
- إدارة التبرعات.
- إدارة الإشعارات.
- إدارة الصلاحيات.
- الاتصال بقاعدة البيانات.

- Notifications
- Payments
